🔐 Challenge Instructions for Participants

Author: Vithula, Venkat Pavan, Agneay Nair

You have been provided with two files:

  • clue.txt – Contains an encoded password.
  • encryptedfile.hc – A password-protected VeraCrypt volume.

🧭 Your Goal

Unlock the VeraCrypt volume to reveal the hidden content inside.

🚀 Steps to Follow

1️⃣ Decode the Password

  • Open the clue.txt file.
  • The contents are encoded using Base64.
  • You can use an online decoder like Base64Decode.org or write a simple decoding script to retrieve the actual password.

2️⃣ Install VeraCrypt (if not already installed)

  • Download it from the official website: https://www.veracrypt.fr/

3️⃣ Mount the VeraCrypt File

  • Launch VeraCrypt.
  • Select an available drive slot.
  • Click Select File and choose encryptedfile.hc.
  • Click Mount.
  • Enter the decoded password when prompted.
  • Once mounted, open the drive to find the Secret Message.

✅ Challenge Complete?

Congratulations! You've successfully unlocked and explored the secret message. 🎉
