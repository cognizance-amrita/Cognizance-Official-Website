Goal: Use Linear Regression to predict student marks based on hours studied.

ğŸ“ 1. Read the Dataset

Use pandas to load the dataset (assume the file is named student_scores.csv).

import pandas as pd
# Load the CSV file
df = pd.read_csv("student_scores.csv")

ğŸ“ Explanation:

We use pandas.read_csv() to load the dataset. The data should have two columns: Hours and Marks.

ğŸ§® 2. Split Data into Features (X) and Target (y)

# Split the dataset
X = df[['Hours']] # Features (2D array for sklearn)
y = df['Marks'] # Target values (1D array)

ğŸ“ Explanation:

We separate the independent variable (Hours) and the dependent variable (Marks). X must be 2D, hence we use double brackets.

ğŸ§  3. Train a Linear Regression Model

from sklearn.linear_model import LinearRegression
# Create the model
model = LinearRegression()
# Fit the model with the data
model.fit(X, y)

ğŸ“ Explanation:

We import LinearRegression from sklearn.linear_model and fit the model using X and y.

ğŸ”® 4. Predict Marks

# Predict for 6 hours of study
predicted_marks = model.predict([[6]])
print(f"Predicted Marks for 6 hours of study: {predicted_marks[0]:.2f}")

ğŸ“ Explanation:

We pass the input in double brackets [[6]] because the model expects a 2D array as input for prediction.

ğŸ“Š 5. (Optional) Evaluate the Model with RÂ² Score

Visit scikit-learn RÂ² Score Documentation to learn more.

from sklearn.metrics import r2_score
# Predict marks for all X
y_pred = model.predict(X)
# Calculate RÂ² Score
r2 = r2_score(y, y_pred)
print(f"Model Accuracy (RÂ² Score): {r2:.2f}")

ğŸ“ Explanation:

The RÂ² score tells us how well the regression line fits the actual data. A value closer to 1 indicates better accuracy.
